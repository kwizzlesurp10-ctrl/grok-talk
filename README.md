# Grok Talk / FusionPanda

A web app for chatting with Grok AI, built with a FusionPanda theme. Live at [grok-talk.vercel.app](https://grok-talk.vercel.app).

## Features

- Chat interface powered by Grok AI
- FusionPanda themed UI
- Progressive Web App (PWA) support
- Mobile-friendly responsive design

## Getting Started

```bash
npm install
npm start
```
